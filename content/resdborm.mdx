# ResDB ORM

ResDB ORM is an Object-Relational Mapping (ORM) tool that simplifies database operations in ResilientDB. It provides an intuitive interface to work with blockchain data using Python classes.

## Installation

```bash
pip install resdb-orm
```

## Quick Start

```python
from resdb_orm import Model, fields
from resdb_orm.connection import connect

# Connect to ResilientDB
connect('http://localhost:8080')

# Define your model
class User(Model):
    name = fields.StringField()
    age = fields.IntegerField()
    email = fields.EmailField()

# Create a new user
user = User(
    name='John Doe',
    age=30,
    email='john@example.com'
)
user.save()

# Query users
users = User.objects.filter(age__gt=25)
for user in users:
    print(f"{user.name}: {user.age}")
```

## Features

- Intuitive model definition
- Automatic schema validation
- Query builder interface
- Transaction management
- Index support
- Relationship handling
